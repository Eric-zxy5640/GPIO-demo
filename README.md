项目概述

本项目旨在帮助 STM32 开发者和学习者更加直观地理解 STM32 GPIO 的工作模式 和 GPIO 寄存器的配置。通过交互式动画与硬件结构图，展示了 GPIO 的不同配置模式和工作机制。这使得开发者能够在实践中更好地理解 STM32 GPIO 的配置和应用场景，提高开发效率。

项目包括以下两部分内容：

GPIO 寄存器配置与工作模式分析(GPIO)：详细讲解了 GPIO 寄存器的功能和如何进行 GPIO 配置。

GPIO 工作模式交互动画(GPIO_MODER)：基于 STM32 GPIO 八种工作模式的交互式动画，帮助用户更好地理解不同模式下 GPIO 的行为。

如何使用

克隆本项目到本地：

git clone https://github.com/Eric-zxy5640/GPIO-demo.git

打开 index.html 文件：
直接在浏览器中打开 index.html 文件，开始浏览交互式 GPIO 配置和硬件结构图。

交互操作：
在页面中点击不同的寄存器模块，查看详细的硬件控制流程和寄存器配置信息。通过交互动画更直观地理解 STM32 GPIO 寄存器的工作原理。
